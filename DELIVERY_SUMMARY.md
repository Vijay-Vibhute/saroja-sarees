## 🚀 Getting Started (3 Steps)

### Step 1: Setup (30 minutes)
```bash
# 1. Install dependencies
npm install @sanity/client @sanity/image-url

# 2. Get API credentials:
# - Sanity: https://manage.sanity.io
# - Razorpay: https://dashboard.razorpay.com (test mode first)

# 3. Create .env file with credentials
REACT_APP_SANITY_PROJECT_ID=your_id_here
REACT_APP_RAZORPAY_KEY_ID=rzp_test_your_key
```

### Step 2: Sanity (1 hour)
```bash
# 1. Create Sanity project
npm create sanity@latest

# 2. Copy schema from sanity-schema/product.ts to your project
# 3. Deploy schema: sanity deploy
# 4. Add test products in Sanity Studio
```

### Step 3: Backend (2-3 hours)
- Choose your framework (Express, Flask, Lambda, etc.)
- Use examples from BACKEND_SETUP.md
- Create 2 endpoints:
  - POST /api/orders/create
  - POST /api/orders/verify
- Deploy backend server

**Total: ~4 hours to live!**
